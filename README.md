# A2_Group11A_MidTermGame

1. Project Title: Hazard
2. Group Number: Group 11A
3. Description:  Following a character with multiple chemical sensitivity (MCS) who falls into a sewer, Hazard begins when the player presess 'Enter' to start the level. On their journey to find the exit, sewage pipes act as walls while coins guide the player's way to the exit. Hazardous chemical monsters must also be avoided to preserve health. 

4. Setup and Interaction Instructions: Players click "I" for instructions and 'Enter' to start playing. Left, Right, Up, and Down arrow keys are used to move in their respective directions as players must avoid monsters and pipe walls to maintian their health in search of the sewer's exit.

5. Iteration Notes

    a. Post-Playtest: Changes made based on playtesting is that we added high-fidelity illustrations/designs to our game, added a way to view               instructions from the main game screen, caption underneath the healthBar along with visual cue for clearer player understanding, added sound, a        cut-scene at the begining of the game, and more monsters to increase the difficulty of the game.
